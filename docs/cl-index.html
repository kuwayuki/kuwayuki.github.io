cluxについて

cluxは、iPhone・iPadからPCをリモート操作できるSSHターミナルアプリです。SSH経由でMac・Windows・Linuxに接続し、tmuxセッションを通じてAI CLIツール（Claude Code等）をリアルタイムに操作できます。

セットアップ手順

  1. PCのSSHサーバーを有効にします（Mac: 「システム設定」→「一般」→「共有」→「リモートログイン」、Windows: OpenSSH Serverを有効化）
  2. cluxアプリの「設定」タブを開きます
  3. PCのIPアドレス（またはホスト名）、ポート番号（通常は22）、ユーザー名、パスワードを入力します
  4. 「接続テスト」ボタンをタップして接続を確認します
  5. 接続に成功したら、ホーム画面からセッションを作成できます

操作のヒント

  • クイックアクションバーからCtrl+CやCtrl+Dをワンタップで送信できます
  • 音声入力を使うと、キーボードなしでコマンドを送信できます
  • 複数のセッションをタブで切り替えて同時に管理できます
  • AI設定からCLIツールの動作モードやWorktreeの有効/無効を切り替えられます
  • セッション一覧を下にスワイプすると、最新のセッション情報に更新されます

よくある質問

Q. SSH接続ができません

デバイスとPCが同じネットワークに接続されているか確認してください。Mac: 「システム設定」→「一般」→「共有」で「リモートログイン」が有効か確認。Windows: 「設定」→「アプリ」→「オプション機能」で「OpenSSH Server」が有効か確認。IPアドレスはネットワーク設定から確認できます。

Q. ターミナルの表示が遅い・文字化けする

ネットワーク環境によって表示に遅延が生じる場合があります。同じネットワーク内で安定したWi-Fi接続をお試しください。文字化けが発生する場合は、PC側のターミナルのエンコーディング設定をUTF-8にしてください。

Q. 音声入力はどの言語に対応していますか？

現在は日本語の音声認識に対応しています。Apple標準の音声認識APIを使用しています。

Q. iPadでも使えますか？

はい。iPhone・iPadの両方に対応しています。

Q. Windowsにも接続できますか？

はい。WindowsでOpenSSH Serverが有効であれば接続できます。Mac、Windows、LinuxなどSSHサーバーが動作するPCに接続可能です。

Q. PC側に何かインストールが必要ですか？

SSHサーバーが有効であれば基本的な操作は可能です。AI CLIツール（Claude Code等）を使用する場合は、PC側にそのツールがインストールされている必要があります。

Q. 外出先から接続できますか？

cluxはローカルネットワーク（同じWi-Fi）内での接続を前提としています。外出先からの接続にはVPNなどの設定が別途必要です。

About clux

clux is a remote SSH terminal app that lets you control your PC from your iPhone or iPad. Connect to Mac, Windows, or Linux via SSH and operate AI CLI tools (such as Claude Code) in real time through tmux sessions.

Setup

  1. Enable SSH on your computer (Mac: System Settings > General > Sharing > "Remote Login". Windows: Settings > Apps > Optional features > "OpenSSH Server")
  2. Open the "Settings" tab in the clux app
  3. Enter your computer's IP address (or hostname), port number (usually 22), username, and password
  4. Tap "Test Connection" to verify the connection
  5. Once connected, you can create sessions from the home screen

Tips

  • Use the quick action bar to send Ctrl+C or Ctrl+D with a single tap
  • Use voice input to send commands without a keyboard
  • Switch between multiple sessions using tabs
  • Adjust CLI tool behavior mode and worktree settings from AI Settings
  • Pull down on the session list to refresh with the latest session information

FAQ

Q. I can't connect via SSH

Make sure your device and computer are on the same network. Mac: check that "Remote Login" is enabled in System Settings > General > Sharing. Windows: check that "OpenSSH Server" is enabled in Settings > Apps > Optional features. You can find your IP address in your network settings.

Q. The terminal display is slow or garbled

Display latency may occur depending on your network conditions. Try using a stable Wi-Fi connection on the same network. If you see garbled text, check that your computer's terminal encoding is set to UTF-8.

Q. Which languages does voice input support?

Currently, Japanese speech recognition is supported. It uses Apple's standard speech recognition API.

Q. Does it work on iPad?

Yes. clux supports both iPhone and iPad.

Q. Can I connect to Windows?

Yes. You can connect to Windows with OpenSSH Server enabled. clux works with any computer running an SSH server, including Mac, Windows, and Linux.

Q. Do I need anything installed on my computer?

Basic operation requires only an SSH server. If you want to use AI CLI tools (such as Claude Code), they need to be installed on your computer.

Q. Can I connect from outside my home?

clux is designed for connections within a local network (same Wi-Fi). Connecting from outside requires additional setup such as a VPN.
